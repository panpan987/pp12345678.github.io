Go语言的数据类型

类型 描述
布尔(bool) true、false
字符串(string) “我是字符串”
数字 见下图
非基本类型 指针、数组、结构体、接口、Map、切片、函数、Channel
数字类型 描述
int8、uint8 -128到127、0到255
int16、uint16 -32768到32767、0到65535
int32、uint32 -2147483648到2147483647、0到4294967295
int64、uint64 -9223372036854775808到9223372036854775807、0到18446744073709551615
float32 1.401298464324817070923729583289916131280e-45到3.402823466385288598117041834516925440e+38
float64 4.940656458412465441765687928682213723651e-324到1.797693134862315708145274237317043567981e+308
byte 类似uint8
rune 类似int32
int 32 或 64 位,取决于操作系统
uint 32 或 64 位,取决于操作系统
